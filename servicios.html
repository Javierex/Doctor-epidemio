Servicios

Podemos ayudarle en cada una de las fases de su proyecto de investigación. Desde la estructura del anteproyecto, hasta el análisis de datos y redacción de discusión. En Doctor Epidemio trabajamos bajos los parámetros de Integridad Científica. Todos los productos que le entregamos se basan en las buenas prácticas de análisis de información y redacción. A continuación encontrará cada uno de los servicios que ofrecemos:

Redacción

Redacción del protocolo de investigación (Introducción, Justificación, Planteamiento del problema y Pregunta de Investigación, Objetivos).

Contenido

Redacción de Marco Teorico/ Estado del Arte.

Metodología

Redacción de metodología (Diseño, Cálculo de la muestra, Descripción de la población, Plan de análisis).

Base de datos Excel

Creación de base de datos excel (Sin ingreso de datos), entrega de diccionario y entrega de formulario fisico en word.

Base de datos Acces

Creación de base de datos Access (Sin ingreso de datos) y entrega de formulario fisico .

Corrección

Corrección de estilo, revisión y edición del protocolo

Encuestas

Aplicación de encuestas.

Historias clinicas

Revisión de historias clinicas.

Inserción de datos

Ingreso de datos a bases de datos

Depuración de datos

Limpieza de la base de datos.

Análisis estadístico univariado

Calculo frecuencias relativas, absolutas, medias, medianas, desviación estandar, etc.

Análisis estadístico bivariado

Pruebas de hipotesis, pruebas de reproducibilidad o concordancia, valores de seguridad prueba diagnostica (sensibilidad, especificidad), etc.

Análisis multivariado

Construcción de modelos; análisis de tiempo al evento, regresión logística, construcción de escalas, curvas ROC, etc.

Búsqueda bibliográfica

Búsqueda bibliográfica para revisiones sitemáticas o para complementar la discusión del proyecto.

Selección de estudios

Selección de estudios para Revisión Sistemática.

Extracción de información

Extracción de información para Revisión Sistemática.

Evaluación de calidad

Evaluación de calidad y sesgos de estudios para Revisiones Sistemáticas.

GRADE

Para Revisiones Sistemáticas.
